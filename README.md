# Labirinto Gráfico - Atividade de Computação Gráfica

Você é um aluno em busca de entrar numa boa faculdade e você escolheu a FEI! 

Após estudar muitas horas para passar pelo vestibulas, você foi escolhido para a próxima fase, um labirinto!

Os candidatos devem concorrer entre si para ver quem é o mais rápido para entrar no portal da FEI. Quem passar primeiro, vence! 

Boa sorte futuro FEIano!

## Como jogar

### Objetivo: Entrar no portal da FEI

### Movimentação do jogador: WASD
### Ativas/Desativas música: Espaço
### Pausar jogo: ESC 

## Pré-requisitos

### clonar o repositório 

```bash 
git clone https://github.com/PauloViniciusAF/ATV1-Computacao-Grafica.git
```

### Criar ambiente virtual

```bash 
python3 -m venv venv
```

### Ativar o ambiente virtual

* Windows

```bash 
venv\Scripts\activate
```

* Linux/Mac
```bash 
source venv/bin/activate
```

### Instalar as bibliotecas

```bash
pip install -r requirements.txt
```
